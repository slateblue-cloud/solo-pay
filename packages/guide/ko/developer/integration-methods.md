# 클라이언트 사이드 연동

SoloPay는 결제 위젯을 통한 클라이언트 사이드 연동 방식을 제공합니다.

## 연동 방식

SoloPay가 제공하는 결제 위젯을 호출하는 방식입니다. 위젯이 결제 생성, 지갑 연결, 서명, 처리를 모두 담당하며, 가맹점은 위젯 호출과 결과 검증만 구현하면 됩니다.

```
가맹점 프론트엔드  ──(위젯 호출)──▶  SoloPay 위젯
                                         │
                              결제 생성 + 지갑 연결 + 결제 처리
                                         │
          │◀──(successUrl/failUrl로 복귀)─│
          │
          │──(paymentId로 결제 최종 확인)──▶  SoloPay API
```

가스리스 서명, Approve 처리, 결제 생성 등 복잡한 Web3 로직을 SoloPay 위젯이 모두 담당합니다. 가맹점은 위젯을 호출하고 결과를 검증하기만 하면 됩니다.
